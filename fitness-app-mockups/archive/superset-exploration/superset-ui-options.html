Recommendation

Option A (Menu Link) + Option B (Combined Card) would work well together:

  • Phase 1: Implement the menu link approach (minimal UI changes, quick to build)
  • Phase 2: Once linked, display as a combined card for better visual grouping
  • Keep the "Add Superset" option in the add menu for power users who know what they want
  • Allow unlinking from the menu to go back to separate exercises

Data Model Consideration

Add a supersetGroupId field to exercises. Exercises with the same ID are in the same superset. The backend stores this and returns it with workout data. This supports 2+ exercises (trisets, giant sets) without schema changes.
